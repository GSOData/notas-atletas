# Projeto Certificação 1 - Notas dos Atletas 🏅

Primeiro projeto de certificação do curso **DEVstart**, focado em consolidar fundamentos de **Lógica de Programação** com JavaScript.

## 📌 Sobre o Projeto
O desafio consistia em criar uma aplicação que recebesse uma matriz de objetos contendo nomes e notas de atletas. A lógica principal do software é:
1. Receber as notas.
2. Ordenar as notas.
3. Eliminar a maior e a menor nota do júri.
4. Calcular a média das notas restantes.

## 🚀 Conceitos Aplicados
Este projeto foi fundamental para praticar:
- **Iteração:** Uso de laços `for` para percorrer matrizes de objetos.
- **Manipulação de Arrays:** Uso dos métodos `.sort()` para ordenação e `.slice()` para extração de subconjuntos.
- **Iteração Funcional:** Uso do `.forEach()` para cálculos aritméticos.
- **Lógica Matemática:** Processamento de dados para chegar a uma média válida.
